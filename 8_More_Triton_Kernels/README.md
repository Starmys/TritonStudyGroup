# More Triton Kernel Examples

## Quick Start

- Requirements
    - 1 NVIDIA GPU
    - CUDA >= 12.1
    - torch == 2.6.0
    - triton == 3.2.0

- Run unit test
    ```bash
    python k_means.py
    python grid_sparse_attention.py
    python stairs_shape_linear.py
    ```
